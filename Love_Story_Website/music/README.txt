HOW TO ADD YOUR ROMANTIC SONG
================================

1. Find your romantic song (MP3 format recommended)
   
2. Rename it to something simple like: romantic-song.mp3
   
3. Copy the song file to this folder:
   d:\Placement Portal\Love_Story_Website\music\

4. Open index.html and find line with <!-- Add your music file here -->

5. Replace the current source line with:
   <source src="music/romantic-song.mp3" type="audio/mpeg">

6. Save the file

7. Test locally by opening index.html in your browser

8. Push to GitHub:
   git add .
   git commit -m "Add romantic background music"
   git push origin main

SONG RECOMMENDATIONS:
- Perfect by Ed Sheeran
- A Thousand Years by Christina Perri  
- All of Me by John Legend
- Thinking Out Loud by Ed Sheeran
- Your favorite romantic Hindi song!

NOTE: Make sure the file is MP3 format for best browser compatibility.
      File size should be under 10MB for faster loading.
